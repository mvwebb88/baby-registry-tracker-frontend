## ğŸ“– Description

Baby Registry is a full-stack web application that allows users to create and manage a personalized baby registry.

Users can:
- Sign up and log in securely
- Set a personalized due date
- Add registry items
- Upload product images
- Categorize and prioritize items
- Mark items as needed or purchased
- Leave comments on items
- View a personalized countdown to their due date

This application was built to simulate a modern, user-centered registry experience while demonstrating full-stack CRUD functionality, authentication, and relational database management.
